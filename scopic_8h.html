Detailed Description

Allows for safe memory operations in C.

This provides tools on allocating memory safely. Provides auto cleaned allocations using macros Supports move operators.

This example shows how to configure unprefixed operators and safely manage allocations using with_func, with_loop, and custom return macros.

Code Example

#include <stdio.h>
#define UNPREFIXED_OPERATORS
#include "scopic.h"
// Wraps malloc inside a tracking function context
void *with_malloc_wrapper(size_t x) with_func (
with_return(malloc(x));
)
// Demonstrates early exit using an untracked value return
int early_exit(size_t x) with_func (
(malloc(x));
with_value_return 2;
)
int main() with_func (
printf("A");
while (1) with_loop (
(with_malloc_wrapper(9999999));
with_break;
);
printf("%i", early_exit(3));
printf("A");
)
